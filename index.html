Financial Modeling


def calculate_roi(revenue, cost):
    """
    Calculate Return on Investment.
    """
    net_profit = revenue - cost
    roi = (net_profit / cost) * 100
    return roi

# Q3 Data
q3_revenue = 5000000
q3_cost = 3500000
print(f"Q3 ROI: {calculate_roi(q3_revenue, q3_cost)}%")

Growth Formula

The compound annual growth rate (CAGR) is calculated as:

\[ CAGR = \left( \frac{EV}{BV} \right)^{\frac{1}{n}} - 1 \]

Where:
$EV$ = Ending Value
$BV$ = Beginning Value
$n$ = Number of years

This formula guides our long-term growth projections.
